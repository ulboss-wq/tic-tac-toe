Документация проекта "Tic Tac Toe"

Цель проекта

Целью данного проекта является создание игры "Крестики-нолики", которая позволяет двум игрокам поочередно делать ходы на игровом поле. Игра реализована с использованием JavaScript для обработки логики игры, манипуляций с DOM и обработки событий. Этот проект демонстрирует использование условных операторов, циклов, функций, массивов и объектов.

Основные функции

  • Игровое поле состоит из 9 ячеек (3x3).
  • Проверка на победу: определяется выигрышная комбинация.
  • Подсчет очков и отображение текущего счета игроков.
  • Обработка ничьей, если все ячейки заполнены и победителя нет.
  • Динамическое обновление интерфейса в зависимости от хода игры.

Технологии

В проекте использованы следующие технологии:

  • HTML: Структура страницы и таблицы игрового поля.
  • CSS: Стилизация элементов, включая игровое поле и кнопки.
  • JavaScript: Логика игры, обработка событий, манипуляции с DOM.
